# Жижиг портфолио (Зөвхөн HTML — Table Layout)

Тодорхойлолт
-----------
Энэ төсөл нь зөвхөн HTML ашиглан (таблaр дээр суурилсан layout) хийсэн жижиг портфолио жишээ юм.
Файлын бүтэц нь нэг `index.html` файлтай бөгөөд аль ч сервэр/сэрвис шаардлагагүй, компьютер дээр браузерээр нээж харахад хангалттай.

Файлууд
-------
- index.html  — Бүх контентыг агуулсан үндсэн HTML файл (table layout).

Хаана, хэрхэн ажиллуулах вэ
--------------------------
1. `index.html` файлыг татаж авна эсвэл шинэ файл үүсгээд доторх кодыг хуулж тавина.
2. Тайлбар: файлд зураг ашиглах бол `img` tag-д зөв зам буюу URL оруулна (жишээ:`src="images/me.jpg"` эсвэл `src="https://..."`).
3. Файлыг хоёр даралт (double-click) эсвэл браузер → Open File хийж нээнэ.

Хэрхэн өөрчлөх вэ
----------------
- Танилцуулга, холбоо, төслүүдийн текстийг `index.html` дээрх тухайн `<td>` эсвэл `<h1>`, `<p>` хэсгүүдэд засна.
- Зураг солих бол `<img src="...">` доторх src-г солих.
- Шинээр проект нэмэх бол Projects бүс дотор nested `<table>`-д шинэ мөр нэмнэ.

Анхааруулга / зөвлөмж
----------------------
- Table-based layout нь хуучны арга бөгөөд responsive (гар утас, жижиг дэлгэц) дэмжлэг муутай. Хэрэв та дараа CSS/modern layout (flex/grid) ашиглах бол сайхан responsive болгох боломжтой.
- Accessibility: Бүх зураганд `alt="..."` нэмэх, таблaр хувилбар дээр semantic-ийг билгэцлэхэд хэцүү тул шаардлагатай бол гарчиг, жирийн текстээр тайлбар нэмнэ.
- Хэрэв та хүсвэл би энэ HTML-г Tailwind/CSS ашиглан responsive болгох код руу хөрвүүлээд өгч болно.

Бусад
-----
Хэрэв та `index.html`-ыг өөрчлөхөд тусламж хэрэгтэй бол өөрийн нэр, зураг URL, төслүүдийн товч мэдээллийг өгөөд хэлээрэй — би таны өгсөн мэдээллээр файлыг шууд засаж өгөх боломжтой.
